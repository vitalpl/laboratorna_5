# laboratorna_5
Репозиторій лабораторної роботи №5 з Конструювання ПЗ.
# Навчальний репозиторій: GitHub Practice

Вітаємо! Це спільний репозиторій для виконання лабораторних робіт.

## Як почати
1. Клонуй репозиторій: git clone <посилання>
2. Прочитай умови лабораторної роботи
3. Створи гілку: git checkout -b feature/твоє-імʼя
4. Виконуй завдання згідно з інструкцією

## Структура
- /students — тут кожен студент створює свій файл
- /shared — спільні файли проєкту

## Правила
- Не пушити напряму в main
- Називати гілки за форматом: feature/імʼя
- Писати осмислені повідомлення комітів
